# Andrews and Arnold CHAOS API

The `chaos` package allows interacting with the [Andrews and Arnold](https://aa.net.uk) CHAOS API.

Implemented features:

* [x] Broadband info
* [x] Broadband quota
* [ ] Broadband PPP kill
* [ ] Broadband ordering
* [ ] Broadband availability checker
* [ ] Broadband ordering
* [ ] Login info
* [ ] Login adjustment
